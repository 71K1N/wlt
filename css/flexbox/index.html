Flex box

Praticando um pouco dos conceitos de Flexbox

Container (flexContainer)

Contem todos os elementos(children)

Exemplo flex 

                    .flex_container{
                        display: flex;
                        background-color: bisque;
                        flex-direction: column;
                    }
                    .children{
                        width: 100px;
                        height: 50px;
                        background-color: aqua;
                        margin: 5px;
                        text-align: center;
                    }
